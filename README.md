# Shopping Cart

A simple shopping cart application built with JavaScript that allows users to add items, update quantities, and calculate the total cost dynamically.

## Features
- Add products to the cart
- Increase or decrease item quantities
- Remove items from the cart
- Automatic total price calculation
- Responsive and straightforward UI

## Tech Stack
- HTML
- CSS
- JavaScript

## Purpose
Built as a practice project to strengthen understanding of JavaScript state management, array operations, event handling, and dynamic DOM updates.
